A professional cost calculation application for Judo classes and training sessions. Built with C# and a structured MVC architecture, KickBlast Judo Cost Calculator helps martial arts schools, instructors, and students accurately calculate training costs with an intuitive and user-friendly interface.

## ğŸš€ Getting Started

### Prerequisites

Before running this application, ensure you have the following installed:

- [.NET SDK](https://dotnet.microsoft.com/download) (version 6.0 or higher recommended)
- Visual Studio 2019/2022 or Visual Studio Code
- Git (for cloning the repository)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ThisumM05/KickBlast_Judo_Cost_Calculator.git
   ```

2. Navigate to the project directory:
   ```bash
   cd KickBlast_Judo_Cost_Calculator
   ```

3. Open the project in your preferred IDE:
   ```bash
   # For Visual Studio Code
   code .
   ```

4. Once the project code is added, restore NuGet packages:
   ```bash
   dotnet restore
   ```

5. Build the project:
   ```bash
   dotnet build
   ```

6. Run the application:
   ```bash
   dotnet run
   ```
