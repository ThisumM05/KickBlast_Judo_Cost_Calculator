# KickBlast Judo Cost Calculator

**KickBlast Judo Cost Calculator** is a desktop application designed to simplify the calculation of fees and costs associated with Judo classes and training programs. It allows users to input class types, duration, and number of participants, and automatically calculates totals with a clear breakdown. This project demonstrates practical use of **MVC architecture**, **C# development**, and **interactive UI design**.

## Features
- Calculate total fees for Judo classes based on multiple inputs.
- Supports different class types and durations.
- Generates a clear cost breakdown for participants.
- User-friendly interface for smooth interaction.
- Demonstrates **Model-View-Controller (MVC)** architecture.
- Fully desktop-based application with responsive UI.
